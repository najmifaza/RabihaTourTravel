

# Rabiha Tour & Travel - Digital Identity & Web Infrastructure

![Status](https://img.shields.io/badge/Status-Developing-yellow?style=for-the-badge)
![Tech](https://img.shields.io/badge/Stack-WordPress%20%7C%20Elementor%20%7C%20Adobe-blue?style=for-the-badge)

Proyek ini merupakan pengembangan identitas digital komprehensif untuk **Rabiha Tour & Travel**, yang mencakup perancangan sistem visual branding, manajemen konten media sosial, dan pengembangan platform informasi paket Umrah/Haji.

## ğŸ›  Technical Stack

### **Design & Creative Engine**
* **CorelDRAW:** Digunakan untuk konstruksi elemen vektor, desain logo, dan layout publikasi digital berkepadatan tinggi. Fokus pada skalabilitas aset tanpa kehilangan fidelitas visual.
* **Adobe Photoshop:** Digunakan untuk optimasi citra raster, *color grading* dokumentasi perjalanan, dan pembuatan *hi-fidelity* mockups untuk konten Instagram.

### **Web Development & Architecture**
* **Engine:** WordPress CMS.
* **Layout Engine:** Elementor Page Builder (Custom Grid System).
* **Optimization:** Fokus pada *Readability Score* dan *Mobile-Responsive Design*.
* **Integration:** WhatsApp API Gateway untuk sistem konversi langsung (CTA).

---

## ğŸš€ Key Implementation Features

### 1. Visual Identity Consistency
Integrasi desain dari format vektor (CorelDRAW) ke format web (SVG/WebP) untuk memastikan identitas brand yang tajam dan konsisten di semua titik sentuh digital.

### 2. High-Efficiency Navigation
Website dirancang dengan prinsip **"Two-Click Information Access"**. Calon jamaah dapat menemukan detail jadwal keberangkatan dan legalitas perusahaan dalam maksimal dua kali interaksi klik dari halaman utama.

### 3. Instagram Content Framework
Pengembangan sistem template desain yang terstruktur untuk mengomunikasikan:
* Informasi Paket Keberangkatan.
* Edukasi Tips Ibadah & Ziarah.
* Social Proof (Dokumentasi Keberangkatan).
